# Componentes de una red

Una red está compuesta por varios componentes esenciales que permiten la comunicación y el intercambio de datos entre dispositivos. El principal componentes es el Host, que es cualquier dispositivo conectado a la red, como computadoras, servidores, impresoras y otros dispositivos inteligentes.

## Dispositivos de red
Los dispositivos de red son hardware que facilitan la conexión y comunicación entre diferentes dispositivos en una red.

### Router

Un router es un dispositivo que dirige el tráfico de datos entre diferentes redes. Conecta redes locales (LAN) a redes más grandes, como Internet, y determina la mejor ruta para enviar los datos.

### Switch

Un switch es un dispositivo que conecta múltiples dispositivos dentro de una red local (LAN). Recibe datos de un dispositivo y los envía al dispositivo de destino correcto dentro de la misma red.

### Access Point (Punto de Acceso)

Un punto de acceso es un dispositivo que permite a los dispositivos inalámbricos conectarse a una red cableada utilizando Wi-Fi u otras tecnologías inalámbricas.

## Medios de transmisión
Los medios de transmisión son los canales a través de los cuales se envían los datos en una red. Pueden ser cableados o inalámbricos. Algunos ejemplos incluyen:
- **Cables de par trenzado**: Comúnmente utilizados en redes Ethernet, consisten en pares de cables de cobre trenzados para reducir la interferencia.
- **Cables de fibra óptica**: Utilizan pulsos de luz para transmitir datos a altas velocidades y largas distancias.
- **Ondas de radio**: Utilizadas en redes inalámbricas, como Wi-Fi y Bluetooth, para transmitir datos sin cables físicos.
