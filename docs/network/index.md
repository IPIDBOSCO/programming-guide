# Redes y comunicación

Hay una pregunta que siempre hago cuando hablo sobre redes:

> ¿Dónde está Internet?

Puede parecer una pregunta extraña, algunos podrían responder "en la nube" o "en los servidores" o incluso "en el router". Pero la realidad es que Internet no está en un solo lugar físico.

Internet es una vasta red de redes interconectadas que abarca todo el mundo. Está compuesta por millones de dispositivos, como computadoras, servidores, routers y otros equipos de red, que se comunican entre sí a través de cables, fibra óptica, satélites y conexiones inalámbricas. Estos dispositivos están distribuidos en diferentes ubicaciones geográficas, desde centros de datos hasta hogares individuales.

En otras palabras, internet es la conexión que te permite ver videos de gatos.

## ¿Qué es una red?

Una red es un conjunto de dispositivos interconectados que pueden comunicarse entre sí para compartir recursos e información. Estos dispositivos pueden incluir computadoras, servidores, routers, switches, impresoras y otros equipos de red.

Las redes pueden variar en tamaño y alcance, desde redes locales (LAN) que conectan dispositivos dentro de un área limitada, como una oficina o un hogar, hasta redes de área amplia (WAN) que abarcan grandes distancias geográficas, como Internet.

Las pueden redes pueden ser cableadas, utilizando cables físicos para la conexión, o inalámbricas, utilizando señales de radio para la comunicación. Además, las redes pueden ser públicas o privadas, dependiendo de quién tenga acceso a ellas.
